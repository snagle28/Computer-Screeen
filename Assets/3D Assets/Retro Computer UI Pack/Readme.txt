Retro Computer UI Pack

Retro Computer UI Pack would be suitable for a variety of projects, from indie horror games to cozy simulators. There are three different styles in the pack: a black and white version, a corporate style in blue colors, and a colorful cute version. 

This UI Pack includes: 

25 icons for desktop and other uses; 

7 windows of different sizes; 

15 elements that can be mixed and matched to create many variations; 

2 types of buttons + input line; 

Mini-player;

4 bubbles for dialogue or tutorials.
